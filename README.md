# cruise-product-search · Claude Code 插件

通过关键词快速搜索邮轮产品，方便用户快速找到心仪的邮轮。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-product-search-claude-plugin
/plugin install cruise-product-search@cruise-product-search-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-product-search**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
